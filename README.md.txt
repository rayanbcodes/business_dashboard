# Small Business Analytics Dashboard

## Overview
Interactive dashboard for small business sales, product performance, and customer analysis. Built with Python, Dash, and Plotly.

## Features
- Sales trend over time
- Top product analysis
- Customer segmentation
- Filter by product category

## Installation
```bash
git clone https://github.com/YOUR_USERNAME/business_dashboard.git
cd business_dashboard
pip install -r requirements.txt
python app.py
